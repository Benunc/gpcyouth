## GitHub Pages

This repo is set up to be published via GitHub Pages. After pushing to GitHub:

1. **Settings → Pages** → Source: Deploy from a branch
2. Branch: `main` (or `master`), folder: `/ (root)`
3. If using a custom domain (e.g. gpcyouth.benandjacq.com), add it under **Custom domain**

The YouTube embed will work once the site is served over HTTPS from your domain (referrer is sent).
